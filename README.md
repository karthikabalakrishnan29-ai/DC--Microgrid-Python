# DC Microgrid Energy Management System (Python)

## Project Overview
This project implements a modular DC microgrid analytical model using Python. 
It includes:

- Layer 1: Microgrid Simulation (PV, Load, Battery SOC)
- Layer 2: Rule-Based Intelligent Energy Management System (IEMS)
- Layer 3: ML-Based Fault Detection (Structural Integration)

## Technologies Used
- Python
- NumPy
- Matplotlib

## Architecture
The system follows a layered and modular design:
- microgrid.py
- iems.py
- fault_detection.py
- main.py

## Objective
To design and implement a function-based DC microgrid control architecture 
without relying on external simulation software.
